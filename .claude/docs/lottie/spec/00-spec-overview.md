# Lottie Spec — Overview

Source: https://lottie.github.io/lottie-spec/latest/

The Lottie Specs documentation is the official reference for Lottie, "a JSON-based format for animated vector graphics." It targets developers creating tools within the Lottie ecosystem by providing detailed specifications about JSON internals.

## Document Requirements

Lottie documents **MUST use JSON** to structure their data. The top-level object is an Animation object. The specification uses RFC 2119/8174 keyword conventions (MUST, SHOULD, MAY, etc.).

## Status

The specification remains a work in progress, documenting approved features from the Lottie Animation Community. Full standardization will be announced once the draft is complete.
